## 🔈 Audio Setup

The app uses your device's built-in text-to-speech to pronounce Malayalam letters and words. **On mobile (Android/iOS) this works automatically.** On a PC/laptop you need to install a Malayalam voice pack first.

### Windows 11
1. Open **Settings** → **Time & Language** → **Speech**
2. Under *Manage voices*, click **Add voices**
3. Search for **Malayalam** and click **Add**
4. Restart your browser

### Windows 10
1. Open **Settings** → **Time & Language** → **Language**
2. Click **Add a language** → search **Malayalam** → click **Next** → **Install**
3. Once installed, go to **Speech** settings and set Malayalam as an option
4. Restart your browser

### macOS
1. Open **System Settings** → **Accessibility** → **Spoken Content**
2. Next to *System Voice*, click **Manage Voices...**
3. Find **Malayalam** and click the download button ⬇️
4. Restart your browser

### Linux (Chrome/Chromium)
Malayalam TTS on Linux depends on your distribution. Chrome on Linux uses the system's espeak or festival engine. Try installing:
```bash
sudo apt install espeak-ng
```
Then in Chrome, go to `chrome://settings/languages` and ensure Malayalam is listed.

### Checking it works
Once installed, open the app, start any lesson, and tap **🔈 hear it** on a letter card. You should hear the Malayalam pronunciation.

> **Note:** If you've installed the voice but still can't hear anything, try restarting your browser completely (not just refreshing the tab).

## 🗣️ Language notes

- This app teaches **colloquial Kerala Malayalam** — the everyday spoken register, not formal/literary Malayalam
- Romanization follows a practical system designed for readability, not strict academic transliteration
